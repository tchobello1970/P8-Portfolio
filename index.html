L'informatique et moi

Initié rapidement aux joies de l'informatique par mon oncle, ingénieur féru de nouvelles technologies, j'ai toujours baigné dans ce milieu.
J'ai croisé des BBC Acorn, Amstrad CPC, Atari ST, Amiga, PC et la joyeuse communauté qui va avec.
Parallèlement à mes activités professionnelles, j'ai régulièrement programmé des adaptations de jeux de société dans différents langages :
Assembleur 68000, Pascal, Actionscript, Python et finalement en PHP/Javascript sur la plateforme Board Game Arena.
Cette dernière activité m'a donné envie de passer à l'étape supérieure : vivre de ma passion.

La formation

  • Lors du cursus universitaire, beaucoup d'algorithmique, de bases de données en compagnie du Pascal, du C++, de l'ASM68k, du Lisp et du Prolog.
  • Une formation web pour maîtriser les nouvelles techniques de développement avec de l'HTML, CSS, MySQL, Javascript, Java...
  • Très récemment, une nouvelle remise à niveau pour développer des sites web responsives et répondre au mieux aux attentes du métier (SEO, Méthode Agile...).

Booki - 2024

L'objectif du projet Booki est d'intégrer l'interface responsive du site avec du code HTML et CSS à partir des maquettes desktop, tablette et mobile réalisées sur Figma tout en respectant les specifications techniques fournies.

Premier projet de la formation, il permet de se familiariser avec l'éditeur Visual Studio Code et le versionning sous Git et Gihub.

Le projet Booki pose les bases d'un environnement de développement performant et la création d'interfaces web responsives est cruciale dans le développement web moderne.

L'objectif du projet Bluel est de créer une page web dynamique pour le site internet d'une architecte d'intérieur en utilisant JavaScript et en communiquant avec une API pour afficher les différents travaux.

Il est également prévu de créer une page de connexion pour l'administrateur du site ainsi qu'une modale permettant d'uploader de nouveaux médias.

La création de pages web dynamiques avec JavaScript, la gestion des interactions utilisateurs et la communication avec des APIs complètent les compétences attendues d'un développeur Front-End moderne.
